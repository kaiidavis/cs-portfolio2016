Starting With HTML

Inside the html at the top, I have it linked to bootstrap, my mainstyle.css and a link to a special font. This code is <link rel="stylesheet" href="https://maxcdn.bootstrapcdn.com/bootstrap/3.3.7/css/bootstrap.min.css" integrity="sha384-BVYiiSIFeK1dGmJRAkycuHAHRg32OmUcww7on3RYdg4Va+PmSTsz/K68vbdEjh4u" crossorigin="anonymous"> <link rel="stylesheet" href="mainstyle.css" type="text/css" /> <link href="https://fonts.googleapis.com/css?family=Faster+One" rel="stylesheet">

Then I added a class to the body to give to the entire page called calculatorbackcolor and the code for it in css is .calculatorbackcolor { background: linear-gradient(315deg, transparent 75%, #d45d55 0)-10px 0, linear-gradient(45deg, transparent 75%, #d45d55 0)-10px 0, linear-gradient(135deg, #a7332b 50%, transparent 0) 0 0, linear-gradient(45deg, #6a201b 50%, #561a16 0) 0 0 #561a16; background-size: 20px 20px; }

Next is making the navbar by making an unordered list with line items for each one and has href to link between the different places I have made. The code for the navbar is <ul id="navbar"> <li> <a href="index.html"> About Me </a></li> <li> <a href="hardware.html"> Hardware </a></li> <li> <a href="howto.html"> How To Page </a></li> <li> <a href="calc.html"> Calculator </a></li> <li> <a href="todolist.html"> Todo List </a></li> <li> <a href="calchowto.html"> Calculator How To Page </a></li> <li> <a href="#"> Vindinium </a></li> <li> <a href="#"> Big Project </a></li> </ul>

To make the header for the page I used a h1 with the thin-red-border class and header class. It has align=center to center the text and it has the heading text of dropdown calculator. The css for the effects made to the heading is .thin-red-border { border-color: #A93226; border-width: 10px; border-style: solid; border-radius: 10px; background-color: white; } .header { font-family: Faster One; color: darkblue; }

A div class of thin-blue-border is made to surround the entire calculator and all of it's contents in a blue border. The css is .thin-blue-border { border-color: #00264d; border-width: 10px; border-style: solid; border-radius: 10px; background-color: white; }

Next I made a span class with an input inside of it to give it a placeholder of answer while having the answer output to the span. It has many classes inputcenter which centers the span, thinner-black-border adds the border around the span, boldtext is to make the outputting numbers bold, calculatorspan changes the width and height to be the same as the input text size, textcalculator is for the font size, and center is used to center the answer. Then there is a </br> to make a space between the answerbox and everything else. The css used for the classes are .inputcenter { position: relative; top: 40%; left: 40%; } .thinner-black-border { border-color: black; border-width: 3px; border-style: solid; border-radius: 3px; background-color: white; } .boldtext { font-weight: bolder; } .calculatorspan { width: 316px; display: block; height: 69px; } .textcalculator { font-size: 40px; } .center { text-align: center; }

The two text input areas are <input> with placeholder 1st and 2nd value, a text type, and id firstvalue and secondvalue. Also class inputcenter1 and 3 is used to move each of them to the respected places, class inputsize is used to make the input box larger, and textcalculator is to make the font larger. The code for this in css is .inputcenter1 { position: relative; top: 40%; left: 20%; { .inputcenter3 { position: relative; top: 40%; left: 36%; } .inputsize { width: 200px; height: 100px; } .textcalculator { font-size: 40px; }

Next is to make a <select> with <option> for each possible option. The select will have class inputcenter2 to put the select in the right place. The class textcalculator is used to change the font. There will be 6 options, addition, subtraction, multiplication, division, exponent, and square root and each of them will have values = +, -, x, /, ^, and âˆš. The last part in the HTML is putting a line break after the second input text box and the button that finds the answer to the equation. The button has the class btn-large which is bootstrap to make the button large. Class buttoncenter centers the button and is set to id = equalsbutton with the button saying Answer. This code looks like </br> <button class="btn-lg buttoncenter" id="equalsbutton">Answer</button>
